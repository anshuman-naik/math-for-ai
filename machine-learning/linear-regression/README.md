# Linear Regression - Foundations

## Concepts Covered

- Covariance / Variance
- Ordinary Least Squares (OLS)
- Normal Equation
- Batch Gradient Descent

## Key Learnings

- Gradient descent converges iteratively to optimal parameters
- Learning rate controls convergence stability
- Loss decreases rapidly initially and plateaus near minimum

## Observations

- High learning rate leads to divergence
- Very low learning rate slows convergence
- Batch GD closely matches normal equation results

## Next Steps

- Stochastic Gradient Descent (SGD)
