# Gradient Descent Variants

## Covered

- Batch Gradient Descent
- Stochastic Gradient Descent (SGD)

## Key Insights

- SGD updates parameters using one data point at a time
- Introduces noise in optimization
- Faster but less stable than batch gradient descent

## Observations

- Loss curve is noisier than batch GD
- Requires smaller learning rate
- Converges near optimal solution but may oscillate
